# CRUD Projects

Repositório com implementações de projetos CRUD (Create, Read, Update, Delete) organizadas por linguagem, e ideias para desenvolvimento futuro. Objetivo: servir como referência prática para técnicas e padrões de operações com dados.

**Linguagem:** Python

**Projetos:** Implementações de projetos CRUD em Python; cada projeto deve incluir um README local com instruções de execução e a lista de dependências.

**Como usar:** Acesse o diretório do projeto em python/<nome-do-projeto>/ e siga o README local. Recomenda-se criar um ambiente virtual e instalar as dependências indicadas.

**Contribuições:** Envie PRs com código testado e um README do projeto. Estrutura sugerida: python/<nome-do-projeto>/
